# Two Variable Problem

***
A **Karnaugh map** can help simplify boolean algebra problems by showing the outputs in a table.
## Examples

### A *and* B

| A/B | 0   | 1   |
| --- | --- | --- |
| 0   | 0   | 0   |
| 1   | 0   | 1   |
### A *or* B
| A/B | 0   | 1   |
| --- | --- | --- |
| 0   | 0   | 0   |
| 1   | 1   | 1   |

### *not* A *and* B
| A/B | 0   | 1   |
| --- | --- | --- |
| 0   | 0   | 1   |
| 1   | 0   | 0   |
### *not* A *and not* B
| A/B | 0   | 1   |
| --- | --- | --- |
| 0   | 1   | 0   |
| 1   | 0   | 0   |
